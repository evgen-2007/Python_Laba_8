Відповідь:
У Python список (list) - це змінна послідовність елементів, що створюється за допомогою квадратних дужок [] і дозволяє додавати, видаляти чи змінювати значення.
Списки доцільно використовувати тоді, коли потрібно змінювати вміст.
tuple) — незмінна послідовність, яка створюється за допомогою круглих дужок (), і після створення її вміст не можна змінити.
Кортежі доцільно використовувати для збереження фіксованих наборів даних, наприклад координат або констант.

Відповідь:
Словник (dict) у Python - це вбудована структура даних, яка зберігає елементи у форматі 'ключ:значення'.
Структура: Словники створюються за допомогою фігурних дужок {}. 
Кожен елемент - це пара, де 'ключ' (унікальний та незмінний тип, напр. рядок) використовується для доступу до 'значення' (може бути будь-якого типу).
Властивості: Словники є змінними (можна додавати/видаляти елементи) та, починаючи з Python 3.7, є впорядкованими (зберігають порядок вставки).
Приклад використання (збереження даних профілю):
"    'username': 'kseniia_zhytnyk',\n"
    "    'role': 'student',\n"
    "    'id': 3\n"
    "}\n"
    "# Доступ до значення за ключем:\n"
    "# print(user_profile['role']) # Виведе 'student'"
